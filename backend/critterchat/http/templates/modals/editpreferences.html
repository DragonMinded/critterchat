Appearance

With this setting set to "light color scheme" light mode will be used no matter what your system setting is. With this setting set to "dark color scheme" dark mode will be used no matter what your system setting is. With this setting set to "system setting" light or dark mode will be chosen according to the rest of your system.
With this setting enabled, any rooms you are in will be ordered above any chats you are in. With it disabled, rooms and chats are ordered purely by last update.
With this setting enabled, messages that are sent from the same person in quick succession will be combined into one block of text. With it disabled, each message from the same person will appear in a separate block.

Notifications

With this setting enabled, you will see a [★] in the title bar when you have unread notifications.
With this setting enabled, audio notifications will play on mobile devices. With it disabled, audio notifications will only play on desktop devices.

Audio Notifications

All notification sounds must be at most kb in size.
